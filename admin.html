⚡ Configurar Firebase
Para que o sistema funcione com links em tempo real, você precisa de um banco de dados gratuito do Firebase. Leva menos de 5 minutos.
1
Acesse console.firebase.google.com e crie um projeto (qualquer nome, ex: canutto-criativos)
2
No menu lateral, clique em Build → Realtime Database → Create Database → escolha uma região → selecione Start in test mode → Enable
3
Clique na engrenagem ⚙️ → Project Settings → role até "Your apps" → clique em </> (Web) → registre o app → copie o objeto firebaseConfig
4
Cole a configuração abaixo e clique em Salvar:
